📌 수업 개요

  • 대상 : 고등학교 2학년
  • 주제 : 패스트푸드 영양 데이터로 배우는 데이터 분석 기초
  • 학습 목표
    1. 데이터 분석 전체 과정 이해 & 흥미 고취
    2. 데이터 정제·통계·시각화 개념을 파인만 기법으로 설명
    3. 구글 시트로 직접 분석 후 결과 시각화
  • 수업 방식 : 파인만 기법(Feynman Technique)
  • 필수 준비물 : 1 인 1 PC, 구글 계정, 가공된 CSV 자료
